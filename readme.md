# HLR Lookup App

PHP HLR Look up app using <a href="https://www.hlr-lookups.com/">hlr-lookups</a> API.

##Requirements

-PHP 5.6
-cUrl
-Composer

## Installation
 Use our Ubuntu Vagrantbox with Ansible : <a href="https://bitbucket.org/wrcx/hlr-boot">HLR Boot</a> or simply clone the repo.
